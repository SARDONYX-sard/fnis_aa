# Developer Documents

## Requirements

- Rust: >= v1.91
- MSVC
- xmake: >= 3.0.6
- Papyrus Compiler (optional — only needed to regenerate prebuilt `.pex` files)
  - Set `SKYRIM_DIR` environment variable, or falls back to
    `D:/STEAM/steamapps/common/Skyrim Special Edition`

## Build

```shell
git submodule update --init --recursive --depth=1
```

```shell
cargo build --release;xmake build -y fnis_aa
```

## Install

Copies the plugin DLL and prebuilt `.pex` files into the output directory.

```shell
xmake install -o ./build/artifact/ fnis_aa
```

Output layout:

```txt
build/artifact/
├── SKSE/
│   └── Plugins/
│       ├── fnis_aa.dll
│       └── fnis_aa/
│           └── config.json     ← generated by d_merge, not by xmake install
└── Scripts/
    ├── FNIS_aa2.pex            ← native stub
    └── FNISVersionGenerated.pex← native stub
```

> **Note:** `config.json` is generated by d_merge, not by `xmake install`.
> See [d_merge](https://github.com/SARDONYX-sard/d_merge) for details.

## Regenerate prebuilt `.pex` files

Prebuilt `.pex` files are committed to the repository under `cxx/papyrus/prebuilt/`
and reused by default. Regeneration is only needed when `.psc` sources are modified.

```shell
xmake build regenerate_pex
```

Requires the Papyrus Compiler. Set `SKYRIM_DIR` to override the default path:

```shell
$env:SKYRIM_DIR=C:/Games/Skyrim Special Edition;xmake build regenerate_pex
```

## Language Server (for `clangd`)

```shell
xmake project -k compile_commands --lsp=clangd --outputdir=build
```

## Change Build Mode

```shell
xmake f -m release
```
